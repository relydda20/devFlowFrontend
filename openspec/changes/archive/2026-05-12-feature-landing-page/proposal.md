# Proposal: DevFlow Landing Page MVP

## Why
DevFlow needs a clear first impression that explains the product fast and gives visitors a direct path into the extension. The landing page should turn the main workflow metrics into a focused marketing message.

## What Changes
Implement the main landing page for DevFlow, an intelligent VS Code extension that analyzes developer workflow patterns in the era of AI-assisted coding.

## Requirements
- The landing page MUST present a clear hero message and primary CTA above the fold.
- The page SHALL include three metric cards for Code Churn, Context Switching, and AI Efficiency.
- The page MUST include a footer with branding, links, and social icons.

## Context
The landing page must clearly communicate the core value proposition of tracking Code Churn, Context Switching, and Agentic Coding Logs. It serves as the primary entry point to convert visitors into users via a direct hero section and call-to-action.
