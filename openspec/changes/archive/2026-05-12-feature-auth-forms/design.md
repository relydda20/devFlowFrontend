# Design System: Auth Forms

## Colors & Theme
- **Inputs**: `bg-slate-900/50` with `border-white/10`. Text should be `text-slate-200`.
- **Input Focus**: Ring color should use the primary accent `#B3C5FF` (`focus-visible:ring-[#B3C5FF]/50`).
- **Labels**: `text-slate-300 text-sm`.
- **Primary Button**: `bg-[#B3C5FF] text-[#0E1322] hover:bg-[#B3C5FF]/80 w-full`.
- **Text Links**: `text-[#B3C5FF] hover:underline` for toggling between Login and Register.
